Demo 11 - Les Inputs / Ouputs

Les inputs et les ouputs sont des mécanismes de communication entre les composants. Les inputs permettent à un composant parent de fournir des données à un composant enfant, tandis que les outputs permettent à un composant enfant d'émettre des évéments vers son composant parent.

Contenu de l'enfant définit dans l'appel du parent
