Composant enfant output

Pour définir un ouput dans un composant enfant, on utilise le décorateur @Output() suivi du nom de la propriété et on crée une instance de la classe EventEmitter pour émettre les événements. Le composant parent peut alors écouter cet événement en utilisant la syntaxe (nomDeLOuput)="MaFonction()" dans le template où il utilise le composant enfant.
