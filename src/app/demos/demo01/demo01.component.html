Demo 01 - Property Binding (One-Way Binding)

Permet d'afficher le contenu d'une propriété directement dans le template HTML.

Affichage d'une variable entre une paire de double accolades.

Unidirectionnel => TS vers HTML

Valeur de maVariable4 (string) : {{ maVariable4 }}

Valeur de maVariable5 (null) : {{ maVariable5 }}

Valeur de maVariable6 (undefined) : {{ maVariable6 }}

Valeur de maVariable7 (any[]) : {{ maVariable7 }}

Valeur de maVariable8 ([string, number]) : {{ maVariable8 }}
