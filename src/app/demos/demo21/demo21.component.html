Demo 21 - Le HttpClient

HttpClient est un module dans Angular qui facilite les appels HTTP vers services externes. Il fournit des méthodes pour effectuer des requêtes : GET, POST, PUT, DELETE, PATCH, ... vers un serveur distant et récupérer les réponses.
