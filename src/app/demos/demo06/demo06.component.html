Demo 06 - Pipes

Les pipes (ou filtres) sont des fonctions qui permettent de transformer les données affichées dans la vue.

Les pipes sont utilisées pour formater, trier, filtrer ou limiter les données affichées.

Les pipes sont utilisés dans les expressions d'interpolation {{ "{{ expression | pipe }}" }} ou dans les directives de liaison de propriété ([prop]="expression | pipe")

Angular fournit un certain nombre de pipes intégrés, tels que : "date", "uppercase", "lowercase", "currency", "decimal", "percent", "async", "json", ...

Vous pouvez également créer vos propres pipes personnalisés (Demo 07)

Exemples :

String

UpperCasePipe

LowerCasePipe

TitleCasePipe

Avant : "Hello World all !"

UpperCasePipe : {{ "Hello World all !" | uppercase }}

LowerCasePipe : {{ "Hello World all !" | lowercase }}

TitleCasePipe : {{ "Hello World all !" | titlecase }}

Number

DecimalPipe

{{ "{{ value_expression | number [ : digitsInfo [ : locale ] ] }}" }}

minIntergerDigits . minDecimalDigits - maxDecimalDigits

Avant : 56123.12345

Après : {{ 56123.12345 | number }}

Après : {{ 123 | number : '4.2-4' }}

Après : {{ 123.02274 | number : '4.2-4' }}

Après : {{ 123 | number : '4.2-4' : 'fr-FR' }}

Date

DatePipe

{{ "{{ value_expression | date [ : format [ : timezone [ : locale ] ] ] }}" }}

Avant : {{ today }}

Après : {{ today | date }}

Après : {{ today | date : 'dd MMMM YYYY HH:mm:ss' }}

Après : {{ today | date : 'dd MMMM YYYY HH:mm:ss' : '+0200' }}

Après : {{ today | date : 'dd MMMM YYYY HH:mm:ss' : '+0200' : 'fr-FR' }}

Après : {{ today | date : 'dd MMMM YYYY HH:mm:ss' : '+0200' : 'fr-FR' | titlecase }}

Percent

PercentPipe

{{ "{{ value_expression | percent [ : digitsInfo [ : locale ] ] }}" }}

Avant : {{ 0.2345 }}

Après : {{ 0.2345 | percent }}

Après : {{ 0.2345 | percent : '2.2-2' }}

Currency

CurrencyPipe

{{ "{{ value_expression | currency [ : currencyCode [ : display [ : digitsInfo [ : locale ] ] ] ] }}" }}

Avant : {{ 12.34 }}

Après : {{ 12.34 | currency }}

Après : {{ 12.34 | currency : 'EUR' }}

Après : {{ 12.34 | currency : 'JPY' }}

Après : {{ 12.34 | currency : 'EUR' : 'symbol' }}

Après : {{ 12.34 | currency : 'EUR' : 'code' }}

Après : {{ 12.34 | currency : 'EUR' : 'code' : '4.2-4' }}

Après : {{ 12.34 | currency : 'EUR' : 'symbol' : '4.2-4' : 'fr-FR' }}

Json

JsonPipe

{{ "{{ value_expression | json }}" }}

Avant : {{ quentin }}

Après : {{ quentin | json }}
